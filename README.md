# InsureAI - Insurance Claims Assistant

Built on Red Hat OpenShift AI by Madhu Ramakrishnappa

## The Problem
Insurance companies lose 6M+ every year due to manual claim processing.
Adjusters spend 7 hours per claim reading documents manually.

## What I Built
An AI pipeline that processes insurance claims automatically using RAG, Summarization, Sentiment Analysis and Information Extraction.

## Features
- RAG Pipeline - Ask questions about claims in plain English
- Text Summarization - 5 bullet point summaries
- Sentiment Analysis - Detects urgency and claimant emotion  
- Information Extraction - Pulls key fields automatically

## Results
- Reduced claim processing time by 93% — from 7 hours to under 30 minutes
- Built on Red Hat OpenShift AI with Granite 3.1 8B
- LangChain + FAISS + HuggingFace Embeddings
